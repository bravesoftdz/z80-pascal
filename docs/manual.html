Introduction

Z80-Pascal is a cross-platform compiler.  That means you write and compile the program in one platform different than the objective one.

Current version doesn't generate executable or object file, it generates an assembler source fiel you must assemble with a Z80 assembler.  The assembler file can be assembled using Pascmo.

Usage

Warning: Read the readme.txt file to know how to get the compiler executable.

Current version works in the form.

z80pas <source_file> <dest_file>

Where source_file is the one with the Pascal program and dest_file is where the result will be saved.
