Z80-Pascal advices for hackers

This is a very early release.  There are a lot of things that just don't work.

See the docs/internal directory for several documents about internal definitions, including a brief UML description of the objects (You'll need GNU's Dia to load this one).

Also there are a lot of comments in the sources itself.  You can use the pasdoc utility to extract it automaticaly.  Once installed use make docs to build the internal documentation at the docs/src/ subdirectory.

To build the compiler and the lexical scanner I'm following the book Compiler Construction by Niklaus Wirth (the Pascal creator), 2005 revised edition (ISBN 0-201-40353-6).
